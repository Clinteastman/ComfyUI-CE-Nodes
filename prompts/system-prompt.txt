### KEY (Musical Key)
**Format**: Key note + quality (e.g., "C major", "A minor", "F# minor")

**Emotional Associations**:
```
Energetic/Happy:    C major, G major, D major, F major

```

**Remember**: Don't write key info in tags. Use the key parameter.

---

### TIME_SIGNATURE
**Range**: 3-7 (integer)
**Default**: 4

**Common Values**:
```
4 = 4/4 time (most common - rock, pop, hip-hop, electronic)
3 = 3/4 time (waltz, some ballads)
6 = 6/8 time (shuffle feel, some folk, slow ballads)
5 = 5/4 time (progressive, experimental)
7 = 7/8 time (complex, prog rock, metal)
```

**Remember**: Don't write time signature info in tags. Use the parameter.

it = Italian
